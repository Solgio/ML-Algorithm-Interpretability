\section{Prodotti attesi}
% Personalizzare definendo i prodotti attesi (facoltativo)
Lo studente dovrà produrre una relazione scritta che illustri i seguenti punti.
\begin{enumerate}
    \item  \textbf{Analisi degli algoritmi di Machine Learning}\\
    Lo studente dovrà analizzare e valutare le qualità dei seguenti algoritmi di Machine Learning:
    \begin{itemize} 
        \item Linear Regression \\
        \item Logistic Regression \\
        \item Support Vector Machine \\
        \item Decision Tree \\
        \item Random Forest \\
        \item XGBoost \\
        \item RuleFit \\
        \item Simbolic Regression \\
    \end{itemize}
        
    \item \textbf{Preparazione dei prompt per LLM} \\
    Per ognuno degli algoritmi analizzati, lo studente dovrà elaborare un prompt per LLM che possa essere utilizzato per migliorare la comprensione e l'interpretabilità dei risultati.
\end{enumerate}

Nel qual caso in cui lo studente, in seguito all'analisi, abbia ancora tempo a sua disposizione è possibile valutare insieme al tutor aziendale l'opportunità di approfondire ulteriormente l'analisi, ad esempio includendo tecniche di semi-supervised learning.
