\section{Contenuti formativi previsti}
% Personalizzare indicando le tecnologie e gli ambiti di interesse dello stage
Durante questo progetto di stage lo studente avrà occasione di approfondire le sue conoscenze nell'ambito del Machine Learning e algoritmica.
\\In particolare, lo studente avrà modo di approfondire i seguenti argomenti:
\begin{itemize}
    \item Comprendere limiti e potenzialità dei principali algoritmi di Machine Learning.
    \item Apprendere nozioni di design di algoritmi di Machine Learning e di interpretabilità dei risultati.
    \item Analizzare come l'\textit{ensamble} di algoritmi possa modificare le prestazioni e la spiegabilità di algoritmi di Machine Learning.
    \item Sviluppare competenze di Prompt Engineering per LLM.
\end{itemize}
